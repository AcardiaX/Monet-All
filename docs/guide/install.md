# 安装

## 注意

::: warning

**刷入前请先确认模块适用的应用版本**

**搞机不谨慎，救砖两行泪**

:::

## 下载

https://monet.jerryz.com.cn/guide/download

## 前提

1. 使用 Magisk 或 KernelSU 对设备进行了root
2. 安卓版本大于等于12
3. 系统支持 Material You

## 安装方法

在 Magisk 或 KernelSU 中打开压缩包，使用音量键选择是否安装对应应用的莫奈取色
