# Monet-All
莫奈取色整合模块

可通过音量键选择是否安装对应应用的取色模块

v1.3更新内容：微信 monet 取色更新 v7.1.0，添加 Bilibili、百词斩、小横条和 Via 的 monet 取色

包含的应用及原作者（来自酷安）：

刷入方式： #Magisk模块#

⚠️⚠️⚠️刷入前请先确认模块适用的应用版本

简介：
莫奈主题是 Android12 的 Material You 新特性，它会根据壁纸来生成系统主题色，软件也会根据系统主题色来变化。
